# Admin Chips 邏輯設定說明
**最後更新時間**：2026-04-21  
這份文件說明了 LoamLab T1 提示詞節點中的「Chips (選項)」在 Admin 後台的邏輯，特別是如何運用「靜默 (Silent)」屬性來達到隱藏與強制帶入的效果。

---

## 1. 一般選項（用戶看得到並可自己換）
這是最正常的使用情境。
- **後台設定**：不勾選【靜默】。
- **效果**：正常顯示在 SketchUp 插件介面上，用戶點選了就會把該字詞加入到 Prompt 裡。
- **策略設定**：
  - 如果選 `替換 (Replace)`：點擊這個選項會取代該區塊內目前的選取值（單選）。
  - 如果選 `追加 (Append)`：點擊這個選項會把字詞接在其他標籤後面（多選）。

---

## 2. 後備預設靜默詞 (Fallback Default) ✨
這個設定能讓你偷偷塞一個「預設備案」給用戶。只要用戶沒有主動在這個節點區塊點擊任何選項，這個隱藏的 Backup 字詞就會默默地帶進去。
- **後台設定**：
  1. 勾選 **【靜默 (Silent)】**
  2. 策略設為 **【單選 / 替換 (Replace)】**
- **效果**：
  - 隱藏不可見。
  - **觸發條件**：因為它是「替換」屬性，如果用戶在介面上隨便點了同節點的其他可見按鈕，這個按鈕就算是成功「替換」掉了這個預設方案。所以**只有在用戶甚麼都沒選的情況下**，它才會發揮作用。
- **應用場景**：如果不選燈光，系統默默加上預設燈光 `natural lighting, daytime`；但如果用戶自己點了 `sunset`，預設光就不介入。

---

## 3. 強制隱藏詞 (Always Append) 💪
這個設定讓你可以在某個節點區塊中，放入無論如何都要生效的核心字串。
- **後台設定**：
  1. 勾選 **【靜默 (Silent)】**
  2. 策略設為 **【多選 / 追加 (Append)】**
- **效果**：
  - 隱藏不可見。
  - **觸發條件**：因為屬性是「追加」，所以不管是用戶在介面上選了哪些選項，或者是什麼都沒選，這個隱藏提示詞**一定會乖乖疊加**進最終發送的參數裡。
- **應用場景**：固定必須加上的畫質詞彙或提示詞黑魔法，如 `best quality, highly detailed` 等，它跟用戶的選擇互不影響。

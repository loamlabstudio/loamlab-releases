# LoamLab 管理員專用：KOL 系統營運 SOP (極簡版)

這是一份給管理員 (您) 看的內部操作手冊，涵蓋了從「新增一位 KOL」到「發放獎金」的完整流程。

---

## 1. 如何新增一位 KOL？ (管理員操作)

當您決定邀請某人成為大使時，請執行以下步驟：

1. **開啟 Supabase 後台**：進入 `users` 資料表。
2. **尋找帳號**：找到該 KOL 的 Email。
3. **開啟權限**：將該筆資料的 `is_kol` 欄位改為 `true`。
4. **自訂代碼**：將 `referral_code` 欄位改為您指定的客製化代碼（例如：`JOHN10`）。
5. **(選用) 設定 Dodopayments 折扣**：
   - 如果這個代碼只是用來追蹤分潤（粉絲買原價），本步驟**可跳過**。
   - 如果您希望粉絲輸入後能打折，請到 Dodopayments 後台的 `Discounts` 新增一個名稱完全相同（`JOHN10`）的折扣碼。

*(完成後，該 KOL 登入插件就會看到專屬的大使面板了！)*

---

## 2. KOL 如何推廣？ (大使操作)

1. KOL 登入 SketchUp 插件。
2. 點擊右上角頭像，進入「大使面板」。
3. 複製頁面上顯示的**專屬代碼**（即您剛才設定的 `JOHN10`）。
4. 將代碼分享給粉絲。

---

## 3. 粉絲如何綁定？ (粉絲操作)

1. 粉絲登入 SketchUp 插件。
2. 進入帳號或付款頁面，找到「大使折扣碼 / 邀請碼」欄位。
3. 輸入 `JOHN10` 並點擊 APPLY。
4. **系統發生什麼事？**
   - 該粉絲會在資料庫與 JOHN **永久綁定**（寫入 `referred_by`）。
   - 未來這位粉絲的每一筆續費，都會算在 JOHN 的業績裡。

## ★ 合夥人 (Partner) 角色說明【內部機密，不對外公開】

合夥人與 KOL 系統架構相同，但分潤比例更高，且**不在任何公開頁面或 UI 揭露此身分**。

### 新增合夥人

步驟與新增 KOL 完全一致，唯一差異：
- 將 `is_kol` 保持 `false`
- 將 `is_partner` 欄位設為 `true`

合夥人登入後，插件會顯示與大使相同的面板（標題不變），但數字反映 Partner 級距。

### 分潤級距對照

| 角色 | Tier 1（1–50人）| Tier 2（51–100人）| Tier 3（>100人）|
|------|---------------|-----------------|----------------|
| KOL  | 5%            | 10%             | 15%            |
| **Partner** | **15%** | **20%**     | **25%**        |

> **注意**：`is_kol` 與 `is_partner` 不應同時為 `true`；系統以 `is_partner` 優先判斷角色。

---

## 4. 自動分潤與發錢 (系統與管理員操作)

1. **系統自動快照**：
   - 粉絲成功付款後，系統的 Webhook 會自動觸發。
   - 系統自動計算 JOHN 現在累積了幾個付費粉絲，判定他處於哪個階梯 (5%, 10%, 15%)。
   - 系統自動將這筆訂單的抽成金額，記錄到 `kol_ledger` (分潤帳本)，狀態標記為 `pending`。
2. **冷卻與對帳**：
   - 為了防止惡意退款，訂單會有 **15 天的冷卻期**。
   - 15 天後，該筆獎金才會轉為「可提領」。
3. **發放獎金 (每月例行公事)**：
   - 管理員每月固定一天，透過後台腳本（或匯出 CSV）拉出所有「可提領」的獎金清單。
   - 使用 PayPal 或銀行匯款手動發錢給 KOL。
